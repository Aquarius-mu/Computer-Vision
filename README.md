# 计算机视觉的第一个作业
## 数据集描述：
数据包含15个场景，每个场景包含200-400张图像，图像的平均分辨率为300*250。
在实验中，从每个场景中选取100张图像作为训练集，另外选择100张图像作为测试集。
词典中单词的数量建议为200、400等100的整数倍，但不要超过500。
* 评价结果需要包括每个场景的**平均分类准确度**、**所有场景的平均分类准确度**、**运行时间**
